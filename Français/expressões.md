# Expressões

-   `C'est normal` Nada demais
-   `Faire les courses` Fazer compras
-   `Au secours!` Socorro!
-   `Pas de problème` Sem problemas
-   `Chut!` Shhhh!
-   `Tous les jours` Todo dia
-   `À l'heure` Na hora
-   `Comment ça?` Como assim?
-   `Ah bon?` É mesmo?
-   `Quoi?` O quê?
-   `Mais non` Claro que não
