## Possessivos

-   Para conjugar o possesivo dos outros casos, adicionar:

| Pessoa    | Nominativo | Acusativo | Dativo | Genitivo |
| --------- | ---------- | --------- | ------ | -------- |
| Masculino |            | -en       | -em    | -es      |
| Neutro    |            |           | -em    | -es      |
| Feminino  | -e         | -e        | -er    | -er      |
| Plural    | -e         | -e        | -en    | -er      |
