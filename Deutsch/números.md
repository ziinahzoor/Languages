# Die Nummern - Números 🚺

**A ordem dos números é invertida em relação a outras línguas. Primeiro as unidades, depois dezenas. Ex.: `fünfzehn` "quinze".**

- `Null` Zero
- `Eins` Um
- `Zwei` Dois
- `Drei` Três
- `Vier` Quatro
- `Fünf` Cinco
- `Sechs` Seis
- `Sieben` Sete
- `Acht` Oito
- `Neun` Nove
- `Zehn` Dez
- `Elf` Onze
- `Zwölf` Doze
- `Zwanzig` Vinte
- `Dreißig` Trinta
