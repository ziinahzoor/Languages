-   É uma língua criada para ser objetiva. Para a individualidade na conversação, também é flexível, permitindo se expressar de formas diferentes dentro da língua.
-   Comumente segue a estrutura SOV para evitar presumir o contexto da frase antes da ação ser descrita.
-   Não costuma-se usar pronomes, que ficam subentendidos pelo contexto da frase. Porém, os pronomes existem e não é considerado estranho utilizá-los.
-   Os verbos usualmente não são conjugados em pessoa, sendo conjugados através da conjugação base do verbo. Porém podem ser conjugados em pessoa.
-   Palavras com conceitos semelhantes no português se tornam uma em Runvarnyr. Ex.: "Frente" e "futuro", são `ful`.
    -   Em situações onde o contexto não deixa claro o significado, pode-se usar uma partícula diferenciativa. Ex.: para `ful`, pode-se usar `ful'lo` para posição e `ful'te` para tempo.
-   O infinitivo termina em `-ah`, sendo o `h` uma espécie de alongamento da vogal para dar a ideia de continuidade.
    -   A conjugação base é diferente do infinitivo, terminando em `-e`.
-   Palavras não têm gênero, mas para explicitar o gênero de substantivos, usa-se os sufixos `-er` para masculino e `-at` para feminino.
-   Verbo = substantivo - última vogal + `-eh`. Adjetivo = substantivo + `-li` ou verbo - `-h` + `-li`.

## Negação

-   A negação se dá pela palavra `ur` antes do que é pra ser negado.

## Interrogação

-   Perguntas são feitas usando a palavra `ver` ou através da entonação na fala.
    -   Se a pergunta já contém uma palavra interrogativa, `ver` é opcional.

## Yrozerna - Casos

### Nominativo

-   O pronome na forma plana é subentendido como o sujeito da frase. Caso seja ambíguo, pode-se usar o sufixo `-r`
-   Quando o sujeito não é explicitado nem na forma plana, subentende-se que é o falante.

### Acusativo

-   Na estrutura SVO, os substantivos não sofrem alterações, sendo a ordem das palavras o que delimita suas funções gramaticais. Porém, para as outras, o objeto deve utilizar o sufixo `-v` para indicar sua função.

## Exemplos da mesma frase em diferentes conjugações

-   SOV: `Oa dav nye`
-   SOV, com o sujeito sendo o falante, implicitamente: `Dav nye`
-   SOV, com o sujeito oculto: `Dav nyoa`
-   SVO: `Oa nye da`
-   SVO, com o sujeito sendo o falante, implicitamente: `Nye da`
-   SVO, com o sujeito oculto: `Nyoa da`
-   SVO, com o sujeito explícito: `Oar nyoa da`
